# Summary

<h2>NEW B-robot EVO <strong>2</strong>:</h2>
 <strong>Extremely</strong> fast, stable and customizable self-balancing robot
The B-robot EVO 2 is an OPEN SOURCE, arduino based self-balancing robot with many features (take a look at the description below)
https://www.youtube.com/watch?v=d6J0ijMG3jI&feature=youtu.be
 Now with battle arms and add-ons to fight another robots (or stand up when you fall).


<h4>Remotely controlled using your <strong>smartphone or tablet</strong> (WIFI).</h4>

This robot is the natural evolution of the <a href="http://www.thingiverse.com/thing:700637">B-robot ORIGINAL</a> and uses the <strong>same electronics and motors</strong> you would need to create the:
<a href="http://www.thingiverse.com/thing:1804534">Air Hockey Robot</a>
<a href="http://www.thingiverse.com/thing:1683764">Sphere-O-Bot (eggbot MOD)</a>
<a href="http://www.thingiverse.com/thing:1590702">iBoardbot</a>
(and more to come...)
Get tired of one robot? <strong>Create another just printing the 3D parts!</strong>



<strong>B-robot EVO 2</strong> is an <strong>Arduino</strong> based robot. An Open software and hardware creation. You can easily <strong>modify its behaviour</strong> changing the acceleration/speed/stability parameters in the Arduino CODE. 

questions? improvement ideas? comments? new add-ons? visit the B-robot <a href="http://forums.jjrobots.com/forum-2.html">community!</a>

# Print Settings

Printer Brand: Ultimaker
Printer: Ultimaker 2
Rafts: Doesn't Matter
Supports: No
Resolution: 0.15-0.2
Infill: 30%

Notes: 
Print the hubs (2) with a 100% infill. Printing everything takes around 6-8 hours
If you prefer to use nytril O-rings instead the GT2 timing belt, print the "Wheel [V11] FOR O-RING (nytrile)" wheels
